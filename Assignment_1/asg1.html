Notes to Grader:

I added extra features (for "awesomness" points) to improve the paint program. These features are:
- Able to specify numerical values for RGB (ranged between 0-255)
- Added transparency, utilizing the alpha value
- Added an undo button to remove the last placed shape, alternative to clearing the entire canvas
- Added a "Rainbow Coloring" button that toggles between normal drawing mode choosing colors with sliders to
rainbow mode which loops around all colors when drawing over time.
